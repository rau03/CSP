HTML/CSS Reference Sheet

Cheat Sheet

HTML Elements

Tag Name Purpose
<a> Anchor Element creates hyperlinks in an HTML document
<head> Head Element contains general information about an HTML page
<title> Title contains a text that defines the title of an HTML document
<tr> Table Row Element used to add rows to a table before adding table data and table headings
<td> Table Data Element can be nested inside a table row element, to add a cell of data to a table
<thead> Table Head Element defines the headings of table columns encapsulated in table rows
<tbody> Table Body Element is a semantic element that will contain all table data other than table heading and table footer content
<th> Table Heading Element used to add titles to rows and columns of a table and must be enclosed in a table row element

CSS Selectors

Attribute Name Purpose
font-family Font Family To change the typeface of text on your webpage.
font-size Font Size To change the size of text on your webpage.
padding Padding The space between the contents of a box and the borders of a box.
margin Margin The space directly outside of the box.
font-weight Font Weight controls how bold or thin the text appears.
letter-spacing Letter Spacing Used to set the horizontal spacing between the individual characters in a element.
